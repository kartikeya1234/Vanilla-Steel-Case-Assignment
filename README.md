# Vanilla Steel Case Assignment
- Create a virtual python environment.
```bash
$ python -m venv NameOfEnv
```
- Activate the environment.
```bash
$ source Path/To/Venv/bin/activate
```
- Install the requirements.
```bash
$ pip install -r requirements.csv
```
